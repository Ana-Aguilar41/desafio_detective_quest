# desafio_detective_quest
Detective Quest é um projeto educacional desenvolvido em C que utiliza uma árvore binária, uma árvore de busca e uma tabela hash para simular a lógica central de um jogo de investigação. O jogador explora a mansão, encontra pistas e relaciona suspeitos, enquanto o código gerencia mapa, pistas e evidências com estruturas de dados fundamentais.
